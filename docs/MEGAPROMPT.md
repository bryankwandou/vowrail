# Vowrail MVP Build Mandate

Build Vowrail as the policy and evidence rail between an autonomous Solana agent and a wallet signature. The product does not promise safe AI. It limits authority through deterministic controls and produces evidence for every decision.

## Required loop

1. A human defines a versioned mandate.
2. The client compiles deterministic policy JSON.
3. An agent intent is decoded and evaluated.
4. The product returns explicit pass or fail checks.
5. Groq may explain the result but cannot change it.
6. A connected wallet anchors the policy hash and decision through Solana devnet Memo.
7. The receipt links to Solana Explorer.

## Acceptance

- Responsive original brand and product site.
- Working policy editor and deterministic simulator.
- Real browser wallet connection and devnet transaction path.
- Server-side Groq integration with input validation and fallback.
- No private key in the client or repository.
- Production build, lint, and typecheck pass.
- Public GitHub repository and Vercel deployment when credentials permit.

No fake TVL, customers, audit badges, mainnet readiness, or animation quota. Motion must explain hierarchy or state.
