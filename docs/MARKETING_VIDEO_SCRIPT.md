# 75-second Product Video

## 0-08s

Visual: agent actions race across a terminal and stop at a hard boundary.  
Voice: “Autonomous agents can spend in seconds. Most wallet permissions still cannot explain what they were allowed to buy.”

## 08-22s

Visual: Policy Studio. Set program allowlist, 5 SOL cap, 25 SOL daily ceiling, and 50 bps slippage.  
Voice: “Vowrail converts an operator mandate into exact rules an agent cannot rewrite.”

## 22-38s

Visual: run a compliant Jupiter intent. Checks turn green one by one.  
Voice: “Every intent is normalized, checked, and given a stable decision hash before the wallet sees it.”

## 38-52s

Visual: connect Phantom, confirm devnet transaction, open Explorer.  
Voice: “Approved decisions become receipt accounts on Solana devnet, linking policy, mandate, provider, amount, and time.”

## 52-65s

Visual: raise amount above 5 SOL. The interface blocks the action.  
Voice: “Change the intent, and the same agent loses authority. Explanation can help an operator understand why. It can never override the rule.”

## 65-75s

Visual: logo, program ID, dashboard, repository.  
Voice: “Vowrail. Let agents act. Keep the mandate human.”
